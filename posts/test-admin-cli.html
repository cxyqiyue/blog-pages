Admin CLI 测试文章

这是通过 admin-cli.py 脚本创建的测试文章。

特性

  • 可以通过 admin.html 在浏览器编辑
  • 导出为 changes.json
  • 通过本地脚本提交到仓库
  • 无需在网页中暴露任何凭证

工作流程

  1. 在 admin.html 中编辑或新增文章
  2. 导出变更为 JSON 文件
  3. 在本机运行 python scripts/admin-cli.py changes.json
  4. 脚本自动提交到 git 并推送

这是一个完全安全的本地工作流程。
