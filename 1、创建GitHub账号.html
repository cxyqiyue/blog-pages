一、创建GitHub账号

1、什么是GitHub？

GitHub 是全球最大的代码托管平台，注册账号后即可创建仓库、参与开源项目并与他人协作。

2、访问GitHub官网

1）GitHub官网：GitHub官网 。

2）如果浏览器打不开GitHub，可以下载Watt Toolkit进行加速。

3）下载 Watt Toolkit ，解压后选择网络加速——GitHub——网络加速，就可以正常访问GitHub了。

3、注册GitHub

1）打开 GitHub注册 进行账号注册即可。

2）填写基本信息

  • Email：输入有效的电子邮箱地址。
  • Password：至少 15 个字符，或不少于 8 个字符且包含数字和小写字母。
  • Username：仅可包含字母、数字或单个中划线，且不能以中划线开头或结尾。
  • Country/Region：选择所在国家/地区。

3）验证邮箱：GitHub 会向你填写的邮箱发送验证邮件，请及时点击邮件中的链接完成验证。

4）验证完后登录账号即可。
